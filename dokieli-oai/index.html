How about an Acid Test?

Assumptions

  • all interactions conform with open standards
  • at least two different interoperable tool stacks
  • free & open access w/ suitable licensing & attribution
  • information is both human and machine-readable
  • interactions without prior out-of-band knowledge

Features

  • WebID+TLS, Web Access Control/ACL, Linked Data Platform
  • W3C Web Annotation Model (replies, peer-reviews, liking, sharing, bookmarking), Activity Streams 2.0 vocabulary
  • W3C Linked Data Notifications for annotations/social sharing
  • Contributions/annotations with Creative Commons license
